Assignment 4 - 11/15/2021

This assignment was fairly simple, compared to the previous one. I understand the status code calls and the http requests/responses better now. I struggled a little at first when implementing the picture as it was not showing up correctly. However, I relaized I was reading the file as a character-set, so that is why. For the other part, I essentially just used conditionals to check for the url and then display the correct page accordingly. It worked, and then I was able to dispaly the 404 correctly when there was no page for it. Overall, the assignment was shorter and I liked it because it helped me to get a better understanding of status codes, the process for creating a server, making it listen, and then sending correct repsonses for what it should be.
